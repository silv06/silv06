## 🔮 소개

지리정보시스템(GIS)과 인공지능을 연결하는 데 특화된 **소프트웨어 엔지니어**이자 **공간 데이터 과학자**입니다. 공간 분석에 대한 전문성과 고도화된 ML/AI 역량을 바탕으로, 확장 가능한 데이터 파이프라인과 실시간 의사결정 지원 프로토타입을 구축하고 있습니다.

- 🚀 **미션:** 복잡한 위성 및 지구 관측 데이터를 실행 가능한 인사이트로 변환하는 자율형 GeoAI 에이전트 개발.
- 💻 **엔지니어링 마인드셋:** 클라우드 인프라 아키텍처 설계부터 직관적인 반응형 웹 프론트엔드까지 아우르는 풀스택 역량.
- 🌍 **전문 분야:** 원격 탐사 데이터 처리, 다중 분광 지수 분석(NDVI, NDWI, BSI) 및 공간 데이터베이스 관리.
- 🎯 **관심 분야:** 대규모 공간 데이터 전처리 자동화, 공간 지능(Spatial Intelligence) 연구 및 오픈소스 GIS 협업.
