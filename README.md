## 소개

경희대학교 지리학과에 재학 중인 **공간 데이터 과학자**이자 **미기후 연구자**입니다. 지리정보시스템(GIS)과 인공지능을 결합한 차세대 GeoAI 연구자를 목표로 하고 있으며, 탄탄한 학문적 기반을 바탕으로 실시간 의사결정 지원 프로토타입을 구축하는 데 집중하고 있습니다. 현재 인공지능학과 복수전공을 준비하며 수학(미적분학)과 프로그래밍 역량을 강화하고 있습니다.

- **미션:** 복잡한 위성 및 미기후 데이터를 실행 가능한 인사이트로 변환하는 자율형 GeoAI 에이전트 개발.
- **엔지니어링 마인드셋:** 클라우드 인프라 기반의 데이터 파이프라인 설계부터 반응형 풀스택 프로토타이핑까지 아우르는 실무 역량.
- **전문 분야:** 원격 탐사, 미기후 분석, 공간 데이터 과학, 그리고 다중 분광 지수(NDVI, NDWI, BSI) 기반 환경 모니터링.
- **관심 분야:** 대규모 공간 데이터 전처리 자동화, 군 IT/개발 및 지형 분석 전문특기병 지원, 오픈소스 GIS 기술 협업.
