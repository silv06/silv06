## 🔮 소개

저는 지리정보시스템(GIS)과 인공지능을 연결하는 데 특화된 **소프트웨어 엔지니어**이자 **공간 데이터 과학자**입니다. 지리학에 대한 탄탄한 기반과 고급 ML/AI 아키텍처 역량을 바탕으로, 강력한 엔드투엔드 데이터 파이프라인과 실시간 의사결정 지원 프로토타입을 구축하는 데 집중하고 있습니다. 

- 🚀 **미션:** 복잡한 위성 및 환경 데이터를 실행 가능한 인사이트로 변환하는 자율형 GeoAI 에이전트 개발.
- 💻 **엔지니어링 마인드셋:** AWS 백엔드 클라우드 인프라 구축부터 반응형 프론트엔드 개발까지 아우르는 풀스택 프로토타이핑 역량.
- 🌍 **전문 분야:** 원격 탐사, 공간 분석(NDVI, NDWI, BSI) 및 환경 모니터링 시스템.
- 🎯 **관심 분야:** 연구 협력, 오픈소스 GIS/AI 프로젝트, 그리고 학제간 융합 제품 개발이 필요한 소프트웨어 엔지니어링 포지션.
